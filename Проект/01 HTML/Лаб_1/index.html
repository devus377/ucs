Напишите заголовок третьего уровня. Создайте параграф длиной не менее 1000 символов. Выделите первое слово ранее созданного параграфа жирным шрифтом, а последнее курсивом. Создайте ненумерованный список городов-миллионников России. Создайте нумерованный список ингридиентов вашего любимого блюда. Создайте ссылки на все ваши профили в социальных сетях. Одна из ссылок должна открываться в новом окне браузера. Возьмите несколько картинок, положите в одну папку с вашим скриптом и отобразите на странице. Задайте ширину картинок (например 150 пикселей). Напишите таблицу, состоящую из четырех столбцов и трех строк. Заполните произвольными данными. Создайте форму. Укажите, что форма будет обрабатываться этим же скриптом. В форме создайте элементы для ввода имени, фамилии, электронного адреса, пароля, а также описание к этим полям. После поля с паролем в форме добавьте элемент выбора пола используя радио кнопки (м, ж). Далее в форме добавьте выпадающий список, состоящий из произвольных элементов. Добавьте ранее созданному списку возможность множественного выбора значений. Ниже в форме добавьте текстовое поле размером 5 строк и 40 колонок. После текстового поля добавьте элемент для прикрепления файла.
