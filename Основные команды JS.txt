Основные команды JavaScript:

Переменные:
let: объявление переменной с возможностью изменения ее значения.
const: объявление переменной с фиксированным значением.
var: устаревшая команда, объявление переменной.
Типы данных:
Number: числовой тип данных.
String: строковый тип данных.
Boolean: логический тип данных (true или false).
Object: тип данных, представляющий объект.
Array: тип данных для хранения коллекции элементов.
Undefined: тип данных, используемый для неинициализированных переменных.
Null: тип данных, представляющий отсутствие значения.
Операторы:
Арифметические операторы: +, -, *, /, %.
Операторы сравнения: ==, ===, !=, !==, >, <, >=, <=.
Логические операторы: && (и), || (или), ! (не).
Условия:
if/else: условный оператор, выполняющий блок кода при выполнении условия.
switch: оператор выбора в зависимости от значения переменной.
Циклы:
for: цикл, выполняющий код определенное количество раз.
while: цикл, выполняющий код до тех пор, пока условие истинно.
do/while: цикл, выполняющий код, а затем проверяющий условие.
Функции:
function: объявление функции с параметрами и блоком кода.
return: оператор, возвращающий значение из функции.
arrow function: сокращенный синтаксис для объявления функции.
Работа с DOM:
document.getElementById('id'): получение элемента по его идентификатору.
document.getElementsByClassName('className'): получение коллекции элементов по их классу.
document.querySelector('selector'): получение первого элемента, соответствующего селектору.
element.innerHTML: получение содержимого элемента в виде HTML или установка нового содержимого.
Работа с массивами:
array.length: получение длины массива.
array.push(element): добавление элемента в конец массива.
array.pop(): удаление последнего элемента из массива.
array.concat(array2): объединение двух массивов.
Работа с объектами:
object.property: доступ к свойству объекта.
object.method(): вызов метода объекта.
Это только основные команды JavaScript. В языке есть множество других команд и возможностей, которые могут быть полезными при программировании.
